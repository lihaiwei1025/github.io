一切重新开始
lihaiwei    2018-09-13    PV 

一直以来都想做个属于自己的网站，可惜自己的前端水平实在是肤浅。真的要准备好好学习前端了！幸好还有伟大的 Google 和 Github，作为面向浏览器写码的程序员，我惊喜的发现可以使用 Github Pages + Hexo 免费搭建个人博客，网络上关于这方面的教程很多。

搭建好博客之后，发现 Hexo 自带的主题太平庸了，于是花了点时间找到了一个简洁美观的主题：fexo ，这套主题我还是比较喜欢的。换了fexo主题之后，我又感觉有的地方太丑，比如文章中代码的显示格式有点乱。本着学习前端的心态，我把作者的代码 clone 下来，从头开始捋。

之前看过一句话 ：Don’t dissect the frog, build it. 意思是只看代码是没啥用的，照着做一遍才能知道具体是怎么回事。所以我参考作者的源码，从头开始，修修改改，大致理解了开发一个 Hexo 主题所需的知识：

  1. 前端基础知识：JavaScipt、HTML、CSS 等。
  2. 模版引擎： Hexo 的博客页面都是通过模板引擎渲染的静态页面。用模板引擎可以比较方便地帮我们实现代码的复用。模板引擎有 swig、ejs、jade 等。
  3. CSS预处理器：CSS 预处理能够帮助我们方便的写 CSS，具有复用代码、函数等功能。常见的 CSS 预处理器有 Sass、Less、 Stylus 等。
  4. Hexo开发文档：包括 Hexo 提供的文档，Hexo 模版，Hexo 变量等。

作者的这个 fexo 主题使用了 ejs 模版引擎，sass 预处理器。我在 fexo 主题的基础之上，重新做了一个功能简陋，勉强能用的主题。还有一些功能尚未实现（评论，归档，搜索等..），等以后慢慢添加。

Whatever，我已经按捺不住，先上线再说！一切重新开始: Hello, World.
